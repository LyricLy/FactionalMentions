# FactionalMentions

You know how you can post roles with different colours in the vanilla game by typing in the corresponding IDs? For example, `[[#9,2]]` gives you Jailor with Coven colours.

It's annoying to remember all those numbers, so what if you didn't have to? That's what FactionalMentions is here for.
You can select the faction you like from the dropdown, or add the name with a hyphen to the end of the role name, like `#jailor-coven`.

<img width="794" height="551" alt="image" src="https://github.com/user-attachments/assets/efc22e27-d99f-41e5-bdb2-f923a861fb18" />

BetterTOS2 is fully supported, as well as different languages and mods that let you change role names. FancyUI's functionality of having customized role icons for different factions functions as well.
Everything should just work how you'd expect. If you find an issue or a mod FactionalMentions isn't compatible with, feel free to contact me or make an issue.
